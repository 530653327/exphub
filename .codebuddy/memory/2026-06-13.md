# 2026-06-13

## 后台新增待办看板 (Kanban) 页面

- 参考开发工坊 kanban design 设计，集成到 ExpHub 后台系统
- **新增文件**:
  - `src/main/java/com/exphub/controller/TodoController.java` — REST API，提供 `/api/todos/kanban`（获取todo_list类型经验列表）和 `/api/todos/{id}/status`（拖放更新状态）
  - `src/main/resources/templates/kanban/index.html` — 看板页面，拖放式 Kanban Board
- **修改文件**:
  - `src/main/resources/templates/layout/base.html` — 侧边栏新增"待办看板"菜单项
  - `src/main/java/com/exphub/controller/PageController.java` — 新增 `/kanban` 路由
- **功能**:
  - 自动从经验库加载 `todo_list` 类型的经验，按状态映射到看板列：ACTIVE→待处理、COMPLETED→已完成、BROKEN/DEPRECATED→待办池
  - 支持拖放操作，拖动卡片到"已完成"列自动将经验状态更新为 COMPLETED
  - 支持搜索筛选和优先级筛选
  - 从 tags 字段提取优先级 (P0/P1/P2/P3)
